Алфавитное кодирование — это способ представления сообщений символами алфавита. 
Алфавитное кодирование используется для сжатия и защиты данных. Схема алфавитного кодирования задается таблицей кодов. 
Префиксная схема — это схема, в которой ни один элементарный код не является префиксом другого. 
Префиксная схема является разделимой (допускает однозначное декодирование).
Коды, исправляющие ошибки — это коды, позволяющие обнаруживать и исправлять искажения. 
Коды, исправляющие ошибки содержат избыточность, которая служит для восстановления информации. 
Расстояние Хэмминга между двумя словами одинаковой длины — это количество позиций, в которых они различаются. 
Кодовое расстояние схемы — это минимальное расстояние между различными кодовыми словами. 
Линейный код — это код, множество кодовых слов которого образует линейное подпространство. 
Линейные коды описываются порождающими матрицами. Линейный код состоит из всех линейных комбинаций строк порождающей матрицы. Циклический код — это линейный код, который вместе с каждым кодовым словом содержит все его циклические сдвиги. Циклические коды реализуются с помощью регистров сдвига. 
Коды Боуза–Чоудхури–Хоквингема — это класс циклических кодов с заданным минимальным расстоянием. Они позволяют исправлять множество ошибок.
Основы криптографии включают шифрование, дешифрование, электронные подписи. Криптография использует теорию чисел и алгебраические структуры. Шифрование с открытым ключом (например, RSA) реализует асимметричное шифрование на основе простых чисел. 
Гамма шифра — это последовательность псевдослучайных чисел, которая используется для поразрядного сложения с сообщением по модулю 2.
