кольцо — это абелева группа по сложению и полугруппа по умножению, причём умножение дистрибутивно относительно сложения слева и справа.
Кольцо называется коммутативным, если для умножения выполнено свойство (A2), то есть умножение коммутативно. 
Кольцо называется кольцом с единицей, если для умножения выполнено свойство (A6), то есть кольцо c единицей — моноид по умножению. 
В кольце нейтральный элемент по сложению называется нулём и обозначается 0, а нейтральный элемент по умножению называется единицей и обозначается 1.
Кольцо — это множество с двумя бинарными операциями (сложение и умножение), удовлетворяющими аксиомам: ассоциативность сложения и умножения, коммутативность сложения, существование нейтрального элемента по сложению (0), существование противоположного элемента, дистрибутивность умножения относительно сложения. 
Кольцо состоит из элементов, для которых выполняются эти аксиомы. 
Поле — это обобщение понятия кольца. 
Поле — это коммутативное кольцо с единицей, в котором каждый ненулевой элемент имеет обратный по умножению. Поле образует структуру, где возможно деление.
поле — это абелева группа по сложению и абелева группа по умножению, исключая 0, причём умножение дистрибутивно относительно сложения
поле — это коммутативное кольцо, которое является коммутативным телом и, стало быть, областью целостности.
Делитель нуля — это ненулевой элемент a кольца, для которого существует ненулевой b такой, что a·b = 0. 
Область целостности — это коммутативное кольцо с единицей, не содержащее делителей нуля. Область целостности является подклассом колец, в котором выполняется правило сокращения.
Идеал — это подмножество I кольца R, замкнутое относительно сложения и умножения на любой элемент из R: для любых a, b ∈ I и r ∈ R, a + b ∈ I и r·a ∈ I. 
Простой идеал — это идеал, обладающий свойством: если a·b ∈ I, то a ∈ I или b ∈ I. 
Максимальный идеал — это идеал, не содержащийся ни в каком другом собственном идеале. Максимальные идеалы порождают поля в факторкольце. Факторкольцо — это множество классов смежности кольца R по идеалу I. Оно строится из исходного кольца и идеала. 
Группа — это множество с одной ассоциативной бинарной операцией, имеющей нейтральный элемент, и каждый элемент обратим. 
Группа порождает подгруппы. 
Полугруппа — это обобщение группы, в которой требуется только ассоциативность (нейтральный элемент не обязателен). 
Подгруппа — это подмножество группы, само являющееся группой. 
Класс смежности — это множество g·H = {g·h : h∈H}. 
Классы смежности связаны с разбиением группы.
Если в полугруппе существует система образующих, состоящая из одного элемента, то такая полугруппа называется циклической
Если в полугруппе нет определяющих соотношений, и все различные слова, составленные из образующих, суть различные элементы носителя, то полугруппа называется свободной.
Теорема Лагранжа состоит порядок конечной группы делится на порядок любой её подгруппы. Эта теорема используется для анализа структуры групп.
Конечное поле — это поле с конечным числом элементов.
 Конечное поле строится как расширение простого поля GF(p). Структура конечного поля определяется его характеристикой (простым числом p) и степенью расширения n: количество элементов равно pⁿ. 
Минимальный многочлен элемента — это многочлен наименьшей степени, корнем которого является данный элемент. 
Минимальный многочлен используется для построения конечных полей.
